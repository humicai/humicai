MarkDown基础语法（MarkDown Basics）

一、基础写法（Basic writing）

1、标题（Headings）

You can create a heading by adding one or more # symbols before your heading text. The number of # you use will determine the size of the heading.
Markdown语法下的标题，你可以在标题文本前添加一个或更多 # 标识，# 标识的数量将决定标题的尺寸。

2、段落（Paragraphs）

Paragraphs in Markdown are just one or more lines of consecutive text followed by one or more blank lines.
Markdown语法下的段落，仅仅只需要一行或多行纯文本即可实现，为了实现段间距，可以紧跟一行或多行空白换行即可。

3、引文（Blockquotes）

You can indicate blockquotes with a >.
Markdown语法下的引文，你可以用 > 标识。

4、样式化文本（Styling text）

You can make text bold or italic.
Markdown语法下的文本样式，你可以用 一个*包裹文本内容让文本倾斜、 用两个则让文本加粗。

二、列表（Lists）

1、无序列表（Unordered lists）

You can make an unordered list by preceding list items with either a or a -.
Markdown语法下的做一个无序列表，可以通过在列表项前面用一个或者-引导完成。

2、有序列表（Ordered lists）

You can make an ordered list by preceding list items with a number.
Markdown语法下的做一个有序列表，可以通过在列表项前面用阿拉伯数字引导完成。

3、嵌套列表（Nested lists）

You can create nested lists by indenting list items by two spaces.
Markdown语法下的做一个嵌套列表，可以通过给下级列表缩进两个空白格的量实现。

三、代码格式化（Code formatting）

1、单行格式化（Inline formats）

Use single backticks (`) to format text in a special monospace format. Everything within the backticks appear as-is, with no other special formatting.
Markdown语法下的单行格式化，使用单引号来格式化文本输出内容显示在一个特定样式区域，在其中的一切内容原样输出显示，没有任何其他特殊格式。

2、多行格式化（Multiple lines formats）

You can use triple backticks (`) to format text as its own distinct block.
Markdown语法下的多行格式化，使用三个引号来包裹自身内容区域。

四、链接（Links）

You can create an inline link by wrapping link text in brackets ( [ ] ), and then wrapping the link in parentheses ( ( ) ).
你可以在中括号中输入文本内容来创建一个链接，并且在后面小括号中输入链接地址。

For example, to create a hyperlink to www.humicai.cn, with a link text that says：轻微生活, you’d write this in Markdown: 轻微生活.
例如，要创建一个跳转到www.humicai.cn的超链接，用一个链接文本标明访问的是轻微生活！在Markdown语法下，可以这样写[轻微生活](http://www.humicai.cn)
